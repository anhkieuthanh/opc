# Directory Structure

## /paperclip/
- `cli/`: CLI tools for project init and management.
- `server/`: API server for the control plane.
- `ui/`: React frontend for dashboard.
- `packages/`: Shared packages in the monorepo.
- `skills/`: Plugin ecosystem and skills manager.
- `evals/`, `tests/`: Testing suites.

## /nanobot/
- `nanobot/`: Core python package and agent loop.
- `webui/`: Development UI.
- `bridge/`, `case/`: Supporting components and examples.
- `docs/`: Extensive documentation.
